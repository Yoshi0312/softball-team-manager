# 手動テストチェックリスト（ログイン→チーム選択→選手登録→成績保存→エクスポート）

## 事前準備
- テスト用アカウントでログイン可能であること。
- テスト対象チームに管理者または編集権限で参加済みであること。
- 既存データを保持したい場合は、先に「データをエクスポート」でバックアップを取得すること。

## 1. ログイン
- [ ] `/login.html` にアクセスし、メール/パスワードでログインできる。
- [ ] 認証後にチーム選択画面（または前回選択チームの画面）へ遷移する。
- [ ] ヘッダーにユーザー名が表示される。

## 2. チーム選択
- [ ] 所属チーム一覧から対象チームを選択できる。
- [ ] 選択後、`index.html` のホーム画面に遷移する。
- [ ] ヘッダーのチーム切替ボタンから再度チーム選択画面へ戻れる。

## 3. 選手登録
- [ ] 「選手一覧を見る」→「選手追加」から必須項目（名前/背番号/メインポジション）を入力して保存できる。
- [ ] 保存後、選手一覧に新規選手が表示される。
- [ ] 追加した選手を再編集し、学年やメモなど任意項目が更新できる。

## 4. 成績保存
- [ ] 「保存済み」または「試合成績」導線から成績入力モーダルを開ける。
- [ ] 試合日・打席結果・投手成績を入力し保存できる。
- [ ] 保存後、成績一覧に対象試合が反映される。
- [ ] 統計ページの年・種別フィルタで保存した試合が集計対象になる。

## 5. エクスポート
- [ ] 設定 > データ管理 で「データをエクスポート」を押下すると JSON ファイルをダウンロードできる。
- [ ] ダウンロードした JSON に `players` / `lineups` / `gameStats` / `settings` が含まれる。

## 6. 破壊的操作の導線確認
- [ ] 「データをインポート」実行時、上書きとバックアップ推奨の確認メッセージが表示される。
- [ ] 「全データを削除」実行時、取り消し不可とバックアップ推奨の確認メッセージが表示される。

## 7. 出欠機能の完了条件（固定）
- [ ] 管理者で「出欠管理」ページを開き、イベントを1件作成できる。
- [ ] 別々の2ユーザーで同一イベントに回答（出席/欠席/保留のいずれか）できる。
- [ ] 回答していないメンバーが「未回答者一覧」に表示される。
